Estructura semantica HTML5

Las etiquetas semánticas son aquellas que son estructuradas

Todas son etiquetas estructuradas, solo las etiquetas <div> no son etiquetas estructuradas

la etiqueta <header> es la etiqueta de cabecera de un sitio web, se usa para escructurar listas de enlaces, funciona para proporcionar información de introducción o de identificación en conjunto con la etiqueta <nav> ambas funcionan para traer contenido de listas dirigidos hacía las partes del sitio web
  • Inicio
  • Servicios

la etiqueta <main> es la etiqueta que define el contenido principal de un sitio web, solo puede existir una etiqueta main por documento

la etiqueta <footer> es la etiqueta que define el pie de página en un sitio web

la etiqueta <nav> es la etiqueta que define un conjunto de enlaces en un sitio web

la etiqueta <article> es la etiqueta que sirve para representar autocontenido

la etiqueta <aside> es la etiqueta que representa un contenido secundario o complementario

la etiqueta <section> es la etiqueta que representa como su nombre lo dice, una sección que representa un contenido genérico

la etiqueta <address> es la etiqueta que representa un dirección, de calle, cra # etc
